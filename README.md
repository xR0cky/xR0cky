### About

I work at the intersection of **web security**, **recon automation**, and **offensive thinking**.  
My focus is not tool hoarding or buzzwords — it is **understanding systems deeply enough to break them cleanly**.

I prefer:
- fewer tools, mastered  
- fewer claims, proven  
- fewer distractions, high execution
